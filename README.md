# HypeSquad Tool

An Electron application to manage your Discord HypeSquad Badges.

## Installation

1. Download this repository and extract it

2. Run `install.bat` (will auto-delete after installation)

## Running the Application

**Run this file**
```bash
run.bat
```

Or manually type this in your terminal:
```bash
npm start
```

## Features

- Login with Discord
- Switch between HypeSquad badges
- View current equipped badge
- tokens are stored locally and safe
